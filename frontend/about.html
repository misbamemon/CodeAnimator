Introduction

Code Animator is a web-based tool to help students visualize sorting and searching algorithms for better understanding using animations.

Purpose

To simplify algorithm learning through real-time visuals and interactive explanations, making it easier for beginners.

Key Features

  • Visualize sorting and searching algorithms
  • Step-by-step code highlighting
  • Interactive explanations
  • User-friendly interface

Future Goals

  • Add graph and tree algorithms
  • Enhance user interactivity
  • Login-based customizations
  • Community feedback system
